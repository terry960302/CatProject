# Cat customize Website

to 디자이너 혜경님

### 사용법

1. 터미널을 켜줍니다.

2. 아래 명령어를 입력해줍니다.
   ```
   npm install
   ```
3. app.js 파일로 들어가서 serviceAccountKey.json파일의 경로를 본인 컴퓨터와 호환되게 변경해줍니다.

4. 저장을 하고 터미널에 가서 아래 명령을 입력해줍니다.
   ```
   npm start
   ```
5. 웹 사이트를 켜고 주소에 `localhost:3000`을 쳐줍니다.

6. 이제 본인이 하셨던 프로젝트가 뜰겁니다.
